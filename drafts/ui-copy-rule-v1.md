# UI Copy Rule v1

## 原则

网站只允许三种表达：

### 1. 一句话理解
不要超过 18 个中文字符

例：
✔ 能力成长地图
✔ 灵感流入口
✔ 模拟人生系统

---

### 2. 三段结构
- 是什么
- 在做什么
- 当前状态

---

### 3. 禁止系统语言

禁止出现在 UI：

- issue
- seed
- data layer
- json字段
- git结构
- draft名称
- v0/v1版本号

---

## UI翻译规则

### drafts → 不展示
### AI Scout → 灵感建议系统
### Human Spark → 我的灵感
### Simulation → 游戏化表达层
### Capability → 能力成长

---

## 输出要求

所有网页必须满足：

- 小于 30% 文本
- 70%视觉/结构
- 每屏只表达一个核心点
