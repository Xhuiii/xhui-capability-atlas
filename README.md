# xhui-capability-atlas

A sustainable, interactive, visual personal capability atlas for portfolio, career transition, self-growth, and long-term asset accumulation.

## Purpose

This repository is used to build and maintain my personal capability system.

It is not only a portfolio. It is a long-term, structured, and interactive archive of my projects, skills, methods, evidence, reflections, and future directions.

## Project Domains

### P0 — Personal Capability Atlas

A visual and interactive system for showing my capabilities, project experience, design methods, innovation output, and career direction.

### P1 — Agent Workbench

A personal AI agent workflow powered by Hermes, Discord, GitHub, Flomo, and future coding agents.

### P2 — AI Product Lab

Small AI product experiments, side-project ideas, and tool prototypes.

### P3 — Content and Personal Brand

Content ideas, personal expression, Xiaohongshu topics, AI-generated media, and personal brand experiments.

### P4 — Designer Shop Lab

Design studio ideas, aesthetic experiments, digital products, and future commercial concepts.

### P5 — Exercise, Health, and Self-Growth

A self-management system for exercise, health data, energy, routines, habits, and future Whoop integration.

## Safety Rules

* No company data
* No confidential project details
* No private credentials
* No sensitive personal documents
* Only manually provided and desensitized materials
* Agent actions require confirmation
* GitHub issues are used for planning and tracking, not for storing secrets
