# xhui-capability-atlas

A sustainable, interactive, visual personal capability atlas for portfolio, career transition, self-growth, and long-term asset accumulation.

## Purpose

This repository is used to build and maintain my personal capability system.

It is not only a portfolio. It is a long-term, structured, and interactive archive of my projects, skills, methods, evidence, reflections, and future directions.

## Project Domains

### P0 — Personal Skill Simulator Atlas / 个人技能模拟器地图

A visual, interactive, and continuously evolving personal skill simulator atlas.

It is not only a portfolio, a resume archive, or a static capability map. It shows proven skills from real projects, visible skills from past work, and possible skills revealed through what-if assumptions, speculative thinking, magical ideas, design concepts, product imagination, playful systems, content experiments, and personal ways of seeing the world.

P0 asks not only:

- What skills have I already shown?
- What capabilities have my projects already proved?

It also asks:

- How do I see the world?
- What assumptions do I naturally create?
- If I could see the shapes of emotions, what would I make?
- If animals could speak, what would they need?
- If a city could sense loneliness, what would change?
- What hidden skills are revealed through these what-if expressions?

P0 treats these what-if ideas not as random fantasies, and not only as concept design. They are skill simulations. They reveal my ability to observe the world, create assumptions, define invisible needs, imagine alternative users, translate abstract feelings into systems, and turn poetic thinking into products, services, interfaces, stories, rituals, tools, games, content, or other possible forms.

P0 can continuously receive inputs from other project domains:

- P1 Agent Workbench provides project management, idea routing, automation, and reflection workflows.
- P2 AI Product Lab provides AI product ideas, high-freedom game concepts, playful systems, interactive narrative, sandbox-style tools, and broader product experiments.
- P3 Content and Personal Brand provides expression experiments, content assets, personal voice, aesthetic identity, and audience feedback.
- P4 Designer Shop Lab provides commercial imagination, design studio ideas, aesthetic experiments, and future productization possibilities.
- P5 Exercise, Health, and Self-Growth provides energy, habits, routines, body-state observations, and long-term execution support.

中文说明：

P0 是一个“个人技能模拟器地图”，不是普通作品集，也不是静态简历库。

它不仅展示我已经知道自己拥有的技能，也展示我通过 what-if 设定、思辨设计、魔法版脑洞、产品想象、内容表达、游戏化系统、个人观察方式等显影出来的潜在技能。

它既包含现实项目里已经被证明的能力，也包含我在不同假设、不同感受、不同观察方式下可能长出来的技能。

比如：
- 如果我能看见情绪的形状，我会做出什么？
- 如果动物会说话，它们会需要什么？
- 如果一座城市能感知孤独，它会发生什么变化？
- 如果记忆可以像天气一样被储存，人们会如何搜索、分享或疗愈它？

这些不是随机幻想，也不只是一种“概念设计”。它们是技能模拟。它们能体现我的观察力、假设能力、隐性需求定义能力、另类用户想象能力、抽象情绪产品化能力，以及把诗意感受转化为产品、服务、界面、故事、仪式、工具、游戏、内容或其他形态的能力。

P0 可以不断接收 P1-P5 的输入，并进一步完善我的个人技能模拟器地图。

## How Domains, Milestones, and Issues Work
## 项目域、里程碑和任务的关系

This repository is organized through three layers:

1. **Project Domains** define long-term directions.
2. **Milestones** define execution stages.
3. **Issues** define concrete tasks.

这个仓库用三层结构管理：

1. **Project Domains / 项目域**：定义长期方向。
2. **Milestones / 里程碑**：定义当前阶段路线。
3. **Issues / 任务卡**：定义具体要完成的事项。

P0-P5 are not meant to be executed equally at the same time. At the current stage, P0 is the main product direction, P1 is the management and automation layer, P2 and P3 are near-term next tracks, P5 is a lightweight support system, and P4 remains in backlog.

P0-P5 不是要同时平均推进。当前阶段，P0 是主线产品方向，P1 是管理与自动化层，P2 和 P3 是近期准备启动的第二梯队，P5 是轻量状态支持系统，P4 暂时进入 Backlog。

## P0 as the Simulator Layer
## P0 作为技能模拟器与汇总层

P0 is the main skill simulator layer of this repository.

It receives structured and unstructured inputs from P1-P5, then transforms them into capability cards, project cards, what-if expression cards, evidence chains, simulation scenarios, and future direction maps.

P0 is not limited to past achievements. It should also include speculative, playful, magical, and imaginative ideas when they reveal taste, system thinking, product imagination, creative direction, or future potential.

P0 是这个仓库的主技能模拟层。

它会接收来自 P1-P5 的结构化和非结构化输入，并把它们转化为能力卡片、项目卡片、What-if 表达卡、证据链、模拟场景和未来方向地图。

P0 不只记录已经完成的成果，也可以收纳更魔法版、更脑洞、更玩乐化的奇思妙想。只要这些想法能体现我的审美、系统思考、产品想象力、创作方向或未来潜力，就可以成为技能地图的一部分。
