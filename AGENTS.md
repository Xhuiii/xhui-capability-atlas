# Agent Instructions for xhui-capability-atlas

This is Tina's personal skill simulator atlas repository.

## Language
- Default language: Chinese.
- Keep code, commands, GitHub labels, file names, and technical identifiers in English when needed.

## Safety
- Do not read or modify files outside this repository unless explicitly approved.
- Do not access company folders, company network, company files, or confidential work materials.
- Do not store secrets in this repository.
- Never print or commit API keys, tokens, auth files, `.env` files, or credentials.
- Treat `~/.codex/auth.json`, `~/.hermes/.env`, and any token file as sensitive.
- Do not ask the user to paste secrets into chat.
- Do not create large changes without first explaining the plan.

## README policy
- The current README is the baseline project definition.
- Do not delete or rewrite existing README explanations.
- Future README changes should be appended to `Update Log / 更新记录`.
- If old ideas become inaccurate, mark them as `Deprecated / 已废弃` or `Refined / 已修正`; do not silently remove them.

## Project meaning
- P0 is Personal Skill Simulator Atlas / 个人技能模拟器地图.
- P0 is not a world simulator and not only a portfolio.
- P1-P5 provide inputs to P0.
- What-if expressions are skill simulations, not random fantasies.
- Concept design is only one possible method of what-if expression, not the full definition.

## Execution style
- Prefer small, reviewable changes.
- Before editing files, summarize the intended change.
- Before committing, show the diff and wait for user confirmation.
- Do not push without explicit confirmation.
